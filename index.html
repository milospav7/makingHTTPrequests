Making an HTTP request

In this mini project I will show you a few different ways for making an HTTP request. We will start from pure AJAX technology using XHR objects, then we will move level up and make requests using Fetch - a feature from ES6 in combination with Promise, and at the very end we will use the latest and one modern way of making requests with Fetch, but now in combination wit Async/Await ES7 feature.

For solid understandig this principles of making requests it is required to have at least a basic knowledge of JavaScript and of course it's newer standard versions such as ES6 and ES7. This is not learning from scratch, it is more like one reminder and all in one comparison for these 3 ways of making HTTP requests.


I have been using an fake API for performing actions with json data.

So, let's get started. Choose one method in navigation menu and follow further instructions.
